# Habit Tracker

A simple, responsive **Habit Tracker** web app built with **HTML, CSS, and Vanilla JavaScript**.  
Track habits, view daily progress, see streaks and completion rates, and visualize performance with a basic chart.

Link: https://habit-tracker-vert-six.vercel.app/
---

## Features
- Add / remove habits with name, frequency, and goal  
- Track habits for the selected date (mark completed)  
- View current streak, habits today, and completion rate  
- Simple chart area to visualize completion rates  
- Responsive layout for desktop & tablet  
- No backend — data stored in browser (**localStorage**)  

---

## Tech Stack
- **HTML5**  
- **CSS3** (Flexbox & Grid for layout)  
- **JavaScript (ES6)**
  
---

##  Usage
### ➤ Add a Habit  
Use the form on the left to add:
- Habit name  
- Frequency (Daily/Weekly/etc.)  
- Goal (e.g., 8 glasses, 30 minutes)

### ➤ Mark Completion  
Mark a habit as completed for the selected date.

### ➤ Change Date  
Use the **Test Date** picker to view or update another day.

### ➤ Track Progress  
The dashboard shows:
- **Current streak**  
- **Habits due today**  
- **Completion rate (%)**
